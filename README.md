## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/CYW-Allen/url-shortener.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

After the installation, now you can run the app by the following command.
  ```sh
  npm start
  ```
